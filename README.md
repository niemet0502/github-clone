
# This Repo contain twitter Clone made with React js


## Installation

* Clone this repo:

```bash
git clone https://github.com/niemet0502/twitter-clone.git
```

* Change directory to the project

```bash
cd twitter-clone
```

* Install dependencies

```bash
npm install or yarn install
```

* launch project`

```bash
npm start
```
